# Plano de Uso Multiusuário - Condomínio CECH

Este documento estabelece as regras de compartilhamento da infraestrutura adquirida pelo Edital 01/2026 para os laboratórios Viva Vox, GEMADELE, ELL, DC e Lab Técnico.

### 1. Formas de Acesso
* **Acesso Remoto:** Os servidores de aplicação serão acessíveis via SSH e VPN para todos os membros cadastrados dos 5 grupos.
* **Acesso Físico:** As estações de trabalho ficam localizadas na sala central do condomínio, com acesso livre em horário comercial.

### 2. Prioridades de Uso
* **Pesquisa Acadêmica:** Prioridade máxima para processamento de dados de teses e dissertações.
* **Ensino:** Uso agendado para práticas de laboratório da disciplina da Profa. Edilayne.
* **Manutenção:** Reservado às segundas-feiras (08:00 - 10:00) para atualizações de segurança.

### 3. Sistema de Agendamento
* O uso das estações físicas e dos recursos de GPU/CPU dos servidores será gerido por um calendário compartilhado no Google Calendar do centro.
* Reservas devem ser feitas com no mínimo 24h de antecedência.

### 4. Responsabilidades
* **Coordenador (Alan):** Gestão de usuários e permissões.
* **Tesoureiro (Wagner):** Fiscalização do estado físico dos equipamentos.
* **Usuários:** Manter o backup de seus dados; o condomínio não se responsabiliza por arquivos salvos fora dos volumes de backup.
